About IMPACT

What is IMPACT?

IMPACT (Independent Metrics for Publication Analysis and Citation Tracking) is an open-source tool that computes rolling 24-month journal impact factors using freely available data from PubMed and NIH iCite, providing a transparent alternative to Clarivate's proprietary Journal Impact Factor.

How is the Rolling IF Calculated?

For each target month, IMPACT computes:

Rolling IF = Citations received in 12-month window / Research articles published in preceding 24-month window

This mirrors the official JIF methodology but uses a rolling monthly window instead of a fixed calendar year, providing more timely and granular metrics.

Data Sources

PubMed E-utilities — Paper discovery, metadata, and publication types.
NIH iCite API — Citation data including cited_by lists for historical reconstruction.

Review-Excluded IF

IMPACT also computes an IF excluding review articles from both the numerator and denominator. This helps detect "review inflation" — when journals boost their IF by publishing highly-cited review articles.

Open Source

All code and data are available on GitHub. Contributions welcome!
