Methodology

Gender Inference

Author gender is inferred from first names using the gender-guesser Python package, which classifies names based on a large international corpus. This is a probabilistic estimate — not self-reported identity.

Limitations

  • Binary classification: Only "woman" and "man" categories are assigned. Non-binary identities are not captured.
  • Name-based inference: Gender is inferred from the name associated with a publication, not the author's actual gender identity.
  • East Asian names: Chinese, Japanese, and Korean names have significantly higher "unknown" rates (~70-80%) because many names are used by multiple genders or are unfamiliar to the classification algorithm.
  • Initials-only names: ~5% of papers have only initials (e.g., "J Smith") and cannot be classified.
  • Confidence threshold: Names classified as "mostly_male" or "mostly_female" are included (probability >= 0.60). Names classified as "androgynous" or "unknown" are excluded.

Citation Analysis

Citation counts are from the NIH iCite Open Citation Collection. Only research articles are included (reviews, editorials, etc. are excluded).

24-Month Rolling Citation Rate

Uses the same monthly rolling formula as the main IMPACT site: for each target month T, count citations received in the 12-month window ending at T by papers published in the preceding 24-month window (months T-36 through T-13), divided by the number of papers. This is computed monthly for each gender pair, controlling for publication age so all papers have the same citation accumulation time.

Cumulative Citations per Paper

Total citations ever received divided by number of papers, by publication year. This metric is biased by paper age — older papers have more time to accumulate citations — but shows lifetime citation impact.

Data Sources

  • PubMed Annual Baseline: ~40M article records with author names
  • iCite Open Citation Collection: Citation links between PubMed articles
  • gender-guesser: Open-source name-to-gender classification
